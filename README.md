# Sneha Mishra Portfolio

This is a recruiter-friendly climate science and policy portfolio built with plain HTML, CSS, and JavaScript for GitHub Pages.

## Contents

- Projects and experience pulled from the resume
- Climate-focused blog section for future posts and research notes
- Skills, education, certifications, and contact information
- Responsive layout for mobile and desktop

## Local Preview

Open [index.html](index.html) directly in a browser, or serve the folder with any static server.

## Updating Content

- Edit project, blog, skills, and education data in [script.js](script.js).
- Update the layout and visual system in [styles.css](styles.css).
- Adjust metadata, navigation, or section copy in [index.html](index.html).

## GitHub Pages

The site is ready for GitHub Pages deployment as a static site. Commit the files and enable Pages on the repository branch you want to publish.
